route()
Jet.router.route()方法可供开发者使用代码进行路由跳转，该方法仅接收一个字符串参数， 该参数应是Jet.router.use() 方法设置的一个存在的路由地址，否则无效。请看一个例子：
<button jon='Jet.router.route("/intro")'>点击跳转到介绍页</button> <script> new Jet(); </script>
以下是输出的html:
1.Jet.prototype.$route()
在Jet方法中，可以使用this.$route()来跳转url，效果与Jet.router.route()一样，也可以跳转到一个http地址。 当跳转目标是http地址时，第二参数是是否设置为新打开一个页面。请看一个例子：
<button jon='jump'>点击跳转到百度</button> <button jon='open'>新标签页打开百度</button> <script> new Jet({ func:{ jump:function(){ this.$route('https://www.baidu.com'); }, open:function(){ this.$route('https://www.baidu.com',true); } } }); </script>
以下是输出的html:
2.Jet.prototype.$route.back
在Jet方法中，可以使用this.$route.back()后退一页
3.Jet.prototype.$route.forward
在Jet方法中，可以使用this.$route.forward()前进一页
