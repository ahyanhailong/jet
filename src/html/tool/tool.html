Jet.$
Jet.$ 对象中有许多常用的功能的函数以及对dom操作的方法，其中的方法多数是从Jetterjs中提取过来的， 详细的使用文档请参考：Jetterjs开发文档。 Jet.$ 中有以下方法可以使用：
cls (根据类名查找元素)
id (根据id查找元素)
tag (根据标签纸查找元素)
attr (根据属性值查找元素)
name (根据name属性查找元素)
select (根据选择器查找元素)
type (判断某一变量的类型)
ct (根据标签名生成一个dom元素)
ajax (发送ajax请求)
load (使用ajax加载一个文件)
html5 (查看是否支持html5)
urlParam (获取url的get参数)
