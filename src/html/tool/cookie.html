$cookie()
Jet中封装了操作cookie方法，在Jet方法中，可以直接使用 this.$cookie() 调用。该方法接受四个参数，当只使用一个参数时，表示从cookie从获取该键对应的值，当时用两个或两个以上参数时，表示设置一个cookie值。 其中第一个参数是键，第二个是值，第三个是过期时间，默认值为1，单位为天，第四个是路径，默认值为 / (全局)。请看一个例子：
键：<input type='text' j='key'/> 值：<input type='text' j='value'/> <button jon='setCookie'>设置cookie</button> <button jon='getCookie'>获取cookie</button> <script> new Jet({ data:{ message:'Hello World' }, func:{ addEle:function(opt){ this.$dom.test_init.html="<span jon='alert(opt.data)' j='message' ></span>"; this.$init() } } }); </script>
