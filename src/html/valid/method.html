Jet.valid
Jet.valid对象中含有很多可以定制的属性的，以下是可用的属性或者方法的列表：(点及下面的列表可以跳转到对应位置)
1.language (设置验证提示信息语言，默认值为 CHINESE，可选值为CHINESE和ENGLISH，不区分大小写)
2.useDefaultStyle (是否使用默认的错误提示样式，默认值为true)
3.showInPlaceHolder (是否将格式要求使用 placeholder 显示，默认值为false)
4.useAlert (是否使用alert方法打印出验证失败信息，默认值为false)
5.init() (初始化验证元素)
6.validate() (代码调用验证表单)
7.validInput() (代码调用验证单个输入元素)
8.addRegExp() (修改或者添加验证的正则表达式)
9.addValidText() (修改或者添加验证失败的提示文字)
10.onOnePass() (单个输入元素验证成功的回调函数)
11.onOneFail() (单个输入元素验证失败的回调函数)
1.language
修改 Jet.valid.language 可以设置验证提示信息语言，默认值为 CHINESE，可选值为 CHINESE 和 ENGLISH，不区分大小写。 请看一个例子：
<div>输入电话号码：<input type="text" jvalid='phone' jdom='lang_dom'></div> <button jon="click:Jet.valid.language='english';Jet.valid.validInput('lang_dom')">设置英文</button> <button jon="click:Jet.valid.language='chinese';Jet.valid.validInput('lang_dom')">设置中文</button> <script> new Jet();//为了初始化 jon 元素 </script>
以下是输出的html:
输入电话号码：
2.useDefaultStyle
修改 Jet.valid.useDefaultStyle 可以设置使用默认的错误提示样式，默认值为 true。请看一个例子：
<div>输入电话号码：<input type="text" jvalid='phone' jdom='default_dom'></div> <button jon="click:Jet.valid.useDefaultStyle=false;Jet.valid.validInput('default_dom')">禁用默认样式</button> <button jon="click:Jet.valid.useDefaultStyle=true;Jet.valid.validInput('default_dom')">使用默认样式</button> <script> new Jet();//为了初始化 jon 元素 </script>
以下是输出的html:
输入电话号码：
一般使用该属性与 onOneFail() 方法相结合来自定义用户自己的错误提示，我们将在下面的内容介绍 onOneFail 的用法。：
3.showInPlaceHolder
修改 Jet.valid.showInPlaceHolder 可以设置是否将格式要求使用 placeholder 显示，默认值为false。请看一个例子：
<div>输入电话号码：<input type="text" jvalid='phone'></div> <button jon="click:Jet.valid.showInPlaceHolder=true">使用placeholder</button> <button jon="click:Jet.valid.showInPlaceHolder=false">禁用placeholder</button> <script> new Jet();//为了初始化 jon 元素 </script>
以下是输出的html:
输入电话号码：
除了上述方式设置 showInPlaceHolder 外，还可以在Jet的生命周期函数中对其进行初始化设置。
4.useAlert
修改 Jet.valid.useAlert 可以设置是否使用alert方法打印出验证失败信息，默认值为false。请看一个例子：
<div>输入电话号码：<input type="text" jvalid='phone' jdom='alert_dom'></div> <button jon="click:Jet.valid.useAlert=true;Jet.valid.validInput('alert_dom')">使用alert</button> <button jon="click:Jet.valid.useAlert=false;Jet.valid.validInput('alert_dom')">禁用alert</button> <script> new Jet();//为了初始化 jon 元素 </script>

5.init
使用 Jet.valid.init() 可以初始化验证元素，适用于使用脚本动态添加dom元素之后对其做初始化操作。请看一个例子：
<div jdom='init_dom'></div> <button jon="click:addInput">添加元素</button> <button jon="click:initInput">初始化元素</button> <script> new Jet({ func:{ addInput:function(){ this.$dom.init_dom.html='输入电话号码：' }, initInput:function(){ Jet.valid.init('init_dom'); //参数可以使用 Jet.DOM 元素 或者 dom元素 //使用参数时是对指定元素的所有子验证元素进行初始化 //也可以不使用参数，表示对所有未初始化的验证元素进行初始化 } } }); </script>

8.addRegExp
使用 Jet.valid.addRegExp() 可以修改或者添加验证的正则表达式。请看一个例子：
请输入四位数字:<input type="text" jdom='regExp_dom' jvalid="fourNumber"><br> <button jon="click:addRegExp">添加验证规则</button> <script> new Jet({ func:{ addRegExp:function(){ Jet.valid.addRegExp('fourNumber',/^\d{4}$/,'请输入四位数字'); Jet.valid.validInput('regExp_dom'); } } }); </script>
以下是输出的html:
请输入四位数字:
9.addValidText
使用 Jet.valid.addValidText() 可以修改或者添加验证的错误提示文字。请看一个例子：
email:<input type="text" jdom='validText_dom' jvalid="email"><br> <button jon="click:addValidText">字符串修改错误提示文字</button><br> <button jon="click:addValidTextJson">json修改错误提示文字</button> <script> new Jet({ func:{ addValidText:function(){ Jet.valid.addValidText('email','修改后的邮箱提示文字'); Jet.valid.validInput('validText_dom'); },addValidTextJson:function(){ Jet.valid.addValidText({ 'email':'json格式修改后的邮箱提示文字' }); Jet.valid.validInput('validText_dom'); //以json格式可以一次修改多个 } } }); </script>
以下是输出的html:
email:

10.onOnePass
使用 Jet.valid.onOnePass() 可以设置单个输入元素验证成功的回调函数。请看一个例子：
不为空:<input type="text" jvalid="notnull"><br> 数字:<input type="text" jvalid="number"><br> <button jon="click:onOnePass">设置验证通过的回调函数</button> <script> new Jet({ func:{ onOnePass:function(opt){ Jet.valid.onOnePass(function(ele){ alert(ele.value); }); opt.ele.innerText='设置成功'; } } }); </script>
以下是输出的html:
不为空:
数字:
11.onOneFail
使用 Jet.valid.onOneFail() 可以设置单个输入元素验证失败的回调函数。请看一个例子：
不为空:<input type="text" jvalid="notnull"><br> 数字:<input type="text" jvalid="number"><br> <button jon="click:onOneFail">设置验证失败的回调函数</button> <script> new Jet({ func:{ onOneFail:function(opt){ Jet.valid.onOneFail(function(ele,err){ alert(ele.value+" : "+err); }); opt.ele.innerText='设置成功'; } } }); </script>
以下是输出的html:
不为空:
数字:
