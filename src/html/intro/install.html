安装使用
Jet可以以script标签引入方式使用或者在lite-server中使用，推荐使用后者。
1.以script标签引入
a.Jet可以以直接在html引入script标签的方式使用，并且支持不在http服务器中运行，不过不在http服务器中使用的话就不支持路由功能和jload属性。在html文件中插入以下代码:
<script src='http://www.theajack.com/jet/assets/js/jet.min.js'></script>
b.可以根据需要选择是否要使用Jet自带的UI框架JUI，JUI同样可以使用script标签插入使用。如需使用，请在HTML文件中插入以下代码
<!-- JUI css --> <link rel="stylesheet" href='http://www.theajack.com/jet/assets/css/jui.min.css'/>
<!-- JUI js --> <script src='http://www.theajack.com/jet/assets/js/jui.min.js'></script>
2.下载并且lite-server中开发
下载安装使用主要分为以下几个步骤：
a.安装nodejs
b.安装lite-server
c.下载Jet并拷贝到lite-server目录下
d.运行
a.安装nodejs
nodejs也就不过多介绍，安装请点击这里
b.安装lite-server
lite-server 是轻量级的仅适用于开发的 node 服务器。以下是被推荐的安装lite-server的方式:
npm install lite-server --save-dev
在你的项目中的 package.json 文件中添加一个 “script” 入口:
"scripts": { "dev": "lite-server" },
lite-server的具体使用笔者不再展开，网上教程很多。
c.下载Jet并拷贝到lite-server目录下
点击这里 下载最新的Jet压缩包，解压后将其中所有的内容拷贝到刚刚新建的lite-server项目的目录下，使用以下命令运行项目，就可以看到一个Hello Jet的示例程序。
npm run dev
d.生产环境中使用
完整项目之后，只需要将 原本的四个文件（如下）放入生产环境中http服务器中即可，值得注意的是配置路由时的路径与根目录等参数必须配置正确。
assets (框架相关的代码和资源，无需用户操作)
src(使用者主要编写代码的地方)
index.html (入口html文件)
index.js (入口js文件，也可将js代码以script标签放到index.html文件中)
